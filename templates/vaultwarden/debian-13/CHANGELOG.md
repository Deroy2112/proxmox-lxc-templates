# Changelog

All notable changes to this template will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.34.3-1] - 2025-01-24

### Added
- Initial Vaultwarden template release
- Pre-built binary from czyt/vaultwarden-binary
- Web vault included
- SQLite database backend
- WebSocket support for live sync
- Systemd service with security hardening
- Default configuration with common options documented
- In-place update support via template-update tool
